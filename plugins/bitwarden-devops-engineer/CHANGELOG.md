# Changelog

All notable changes to the bitwarden-devops-engineer plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-04-08

### Added

- `bitwarden-devops-engineer` agent for advisory questions about workflow compliance, linter rules, and action security

## [0.1.0] - 2026-03-19

### Added

- Initial release of the bitwarden-devops-engineer plugin
- `/workflow-fix` command for GitHub Actions workflow linter compliance
- `/action-audit` command for org-wide action usage auditing and remediation
- `bitwarden-workflow-linter-rules` skill with reference knowledge for all 10 linter rules
