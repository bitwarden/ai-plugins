description: Retrieve institutional knowledge for the current repository before starting a code review
---

Invoke the recalling-review-knowledge skill to retrieve past learnings for this repository.

**Usage:**
- `/advise-review` - Load knowledge for current repository

The skill will:
1. Detect the current repository
2. Load institutional knowledge if it exists
3. Present failed detections, repository gotchas, and effective methodologies
4. Provide actionable guidance for your review

**Tip**: Run this before starting any code review to benefit from past experience.
