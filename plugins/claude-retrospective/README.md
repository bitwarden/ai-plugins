# Claude Retrospective Plugin

Comprehensive analysis of Claude Code sessions to identify successful patterns, problematic areas, and opportunities for improvement.

## Features

Analyze Claude Code sessions to extract insights, document learnings, and identify improvement opportunities.

**Skills**:
- `retrospecting` - Comprehensive session analysis
- `extracting-session-data` - Parse session logs and extract structured data
- `analyzing-git-sessions` - Analyze git commits and changes within timeframes

---

## Configuration

### Settings

No special configuration required. The skills work out of the box.

### Requirements

- Claude Code with plugin support
- Git repository with history
